# Hubcredo Automation Platform

A full-stack authentication and automation dashboard system built for the Hubcredo assignment.

## Features
- User Authentication (Sign Up/Login)
- Dashboard with Welcome Message
- Automation Workflow Engine
- Loop Orchestration Controls
- System Logs & Analytics
- Real-time Monitoring

## Tech Stack
- Frontend: React, Tailwind CSS
- Backend: Node.js, Express
- Database: Firebase
- Deployment: Render

## Live Demo
https://hubcredoautomationbynauman.onrender.com

## Local Setup
1. Clone repository
2. Install dependencies: `npm install`
3. Add Firebase config in `.env`
4. Run: `npm start`
